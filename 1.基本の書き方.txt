#include <stdio.h>

int main(void) {
    //表示するためにprintf関数を使用
    printf("Hello World!");

    return 0;
}

各行の説明

#include <stdio.h>
#includeと記述することによって、そのあとに書かれた<>に囲まれたhファイルの中に定義されている関数をこの行以降で呼び出して実行できるようにします。
また、<>の中に書かれたhファイルをライブラリと言います。

int main(void) {}
main関数を定義します。main関数はC言語の標準規格によりコンピューターが実行するときの最初の入り口「エントリーポイント」であると決められています。
{}の中に上から順に処理したい内容を記述していきます。

//
これを記述した行はコメントとして扱われ、コンパイル時には削除されます。
プログラム上のメモみたいなものです。

return 0
実行元であるコンピューターに正常にプログラムが終了したことを通知するためにこれを書きます。
returnの後の数字を1にすると、異常終了したことをコンピューターに通知できます。
